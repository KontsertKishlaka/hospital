# Веб-приложение «Пациент поликлиники»

Данное приложение разработано на стеке ASP.NET Core MVC 8.0 и MS SQL Server. Оно позволяет пациентам регистрироваться, авторизовываться и управлять своими медицинскими записями.

## Функциональные возможности

- **Авторизация и Регистрация**: Полноценная система аутентификации на базе ASP.NET Core Identity, интегрированная с таблицей `Patient`.
- **Личный кабинет**: Главная страница с быстрым доступом ко всем функциям.
- **Профиль пациента**: Просмотр и редактирование личных данных (ФИО, телефон, адрес, дата рождения, пол).
- **Запись на услугу**: Выбор медицинской услуги и времени приема.
- **Предстоящие записи**: Список активных записей к врачу.
- **История посещений**: Просмотр завершенных приемов и их стоимости.

## Технологический стек

- **Backend**: C#, ASP.NET Core 8.0 MVC.
- **Database**: MS SQL Server, Entity Framework Core (Code First / Database First).
- **Security**: ASP.NET Core Identity.
- **Frontend**: Razor Pages, Bootstrap 5, jQuery.

## Инструкция по развертыванию

1. **База данных**:
   - Убедитесь, что у вас установлен MS SQL Server (или SQL LocalDB).
   - Выполните SQL-скрипт `docs/tech/script.sql` для создания структуры базы данных.
   - База данных по умолчанию называется `Hospital_BukarevBedin_2ISP11-41`.

2. **Настройка подключения**:
   - Откройте `appsettings.json`.
   - Проверьте строку подключения `DefaultConnection`. По умолчанию используется `(localdb)\mssqllocaldb`.

3. **Запуск приложения**:
   - Откройте проект в Visual Studio 2022 или VS Code.
   - Выполните `dotnet restore` для восстановления пакетов.
   - Запустите приложение с помощью `dotnet run` или `F5`.

4. **Первое использование**:
   - Перейдите на страницу регистрации.
   - Заполните все поля (телефон должен быть уникальным, так как он используется в логике поиска).
   - После регистрации вы будете автоматически авторизованы и перенаправлены в личный кабинет.

## Структура проекта

- `Controllers/`: Логика обработки запросов (Account, Home, Patient, Appointments).
- `Models/`: Сущности базы данных и ApplicationUser.
- `ViewModels/`: Модели данных для представлений (форм и списков).
- `Data/`: Контекст базы данных `HospitalDbContext`.
- `Views/`: Razor-шаблоны интерфейса.
